# Nepse AI Analyst

A sophisticated stock analysis and portfolio management application tailored for the Nepal Stock Exchange (NEPSE). Built with React, Tailwind CSS, and powered by Google's Gemini AI.

## Features

### 🚀 Market Intelligence
- **Real-time Sentiment Analysis**: AI-driven market pulse and sentiment tracking.
- **Top Movers & Most Active**: Instant view of market leaders and high-volume assets.
- **Sector Heatmap**: Visual representation of sector performance.

### 📊 Advanced Analysis
- **AI Stock Analysis**: Deep-dive analysis of individual stocks using Gemini AI.
- **Technical Indicators**: Interactive charts with Volume, EMA, SMA, Bollinger Bands, RSI, and MACD.
- **Floor Sheet Data**: Track top buying and selling brokers for any stock.
- **Stock Comparison**: Side-by-side comparison of multiple assets.

### 💼 Portfolio & Watchlist
- **Comprehensive Portfolio Tracking**: Monitor holdings, current value, total investment, and P/L.
- **Interactive Performance Charts**: Visualize portfolio history and allocation.
- **Smart Watchlist**: Track your favorite stocks with mini sparkline charts for quick trend analysis.

### 🤖 AI-Powered Tools
- **AI Vision**: Analyze charts and financial documents via image upload.
- **AI Studio**: Generate market visions and conceptual visualizations.
- **Deep Research**: Perform extensive research on market policies and economic impacts.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Framer Motion, Recharts.
- **Backend**: Express (for API simulations and server-side logic).
- **AI**: Google Gemini API (Pro and Flash models).
- **Database/Auth**: Firebase.

## Getting Started

1. **Clone the repository**
2. **Install dependencies**: `npm install`
3. **Set up environment variables**: Create a `.env` file based on `.env.example` and add your `GEMINI_API_KEY`.
4. **Run the development server**: `npm run dev`

## Deployment
This app is designed to be deployed on Cloud Run or similar containerized environments.
